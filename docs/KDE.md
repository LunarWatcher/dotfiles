# KDE post-setup steps

## KDE

Activate `Profile 1`. Everything else is automated.

## Plasma

Plasma itself is somewhat resistant to automation, so plasma itself has post-install steps

### Keybinds

Navigate to shortcuts, then import scheme. KDE's builtin defaults are much weaker than those of Cinnamon, so this is a required step.

The builtin shortcuts are the core shortcuts, while the other shortcut files are various command shortcuts.

### Clipboard history

Clipboard history is enabled by default. It can be disabled via the clipboard item in the tray.
